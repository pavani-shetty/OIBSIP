# Task 3 — Temperature Converter

**Track:** Web Development & Design  
**Level:** 1  
**Task:** Temperature Converter Website

## Objective

Build an interactive web tool that converts temperature between Celsius, Fahrenheit, and Kelvin with input validation.

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Feature Checklist

- [ ] Numeric input field with validation (reject non-numeric input)
- [ ] Input unit selector (Celsius / Fahrenheit / Kelvin)
- [ ] Output showing all converted units
- [ ] Convert button triggers calculation
- [ ] Result display with correct unit labels
- [ ] Edge case: message for below absolute zero (−273.15°C)
- [ ] Clean, centred UI layout

## Conversion Formulas

- C to F: `(C × 9/5) + 32`
- F to C: `(F − 32) × 5/9`
- Kelvin: add/subtract 273.15

## How to Run

Open `index.html` in your browser.
