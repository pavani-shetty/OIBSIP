## Project Overview

A modern, elegant personal portfolio website for **Pavani Shetty** — a final-year Web Developer. Features a dark theme with purple-pink gradients, smooth scroll navigation, and fully responsive layout.

## Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- Google Fonts (Cormorant Garamond + Plus Jakarta Sans)

## Sections

- **Hero** — Name, role, avatar, description, CTA buttons
- **About** — Background and interests
- **Skills** — 6 skill cards with hover effects
- **Projects** — Landing Page & Temperature Converter with GitHub links
- **Resume** — Education, skills, certifications, achievements & PDF download
- **Contact** — Email, phone, LinkedIn, GitHub with social buttons
- **Footer**
