# Task 2 — Personal Portfolio

**Track:** Web Development & Design  
**Level:** 1  
**Task:** Personal Portfolio

## Objective

Create a personal portfolio website showcasing your skills, projects, and contact information.

## Tech Stack

- HTML5
- CSS3
- JavaScript (optional)

## Feature Checklist

- [ ] Profile/hero section: name, role title, photo or avatar
- [ ] About Me section (2–3 sentences)
- [ ] Skills section (visual grid or list)
- [ ] Projects section (at least 2 project cards)
- [ ] Contact section (name, email, LinkedIn/GitHub)
- [ ] Smooth scroll navigation between sections
- [ ] Consistent branding (colours, fonts)
- [ ] Fully responsive (desktop + mobile)

## How to Run

Open `index.html` in your browser.
