# Task 1 — Landing Page

**Track:** Web Development & Design  
**Level:** 1  
**Task:** Landing Page

## Objective

Build a visually polished static landing page for a product, service, or brand of your choice.

## Tech Stack

- HTML5
- CSS3 (no JavaScript required)

## Feature Checklist

- [ ] Fixed or sticky navigation bar with at least 3 navigation links
- [ ] Hero section with headline, subheadline, and CTA button
- [ ] At least 2 distinct content sections (Features, About, Testimonials, etc.)
- [ ] Footer with contact/social links
- [ ] Consistent colour palette across all sections
- [ ] Responsive layout (Flexbox or Grid)
- [ ] No element overlap — intentional padding, margin, box-sizing
- [ ] Clean typography with at least 2 font sizes (heading vs body)

## How to Run

Open `index.html` in your browser.
