# Task 1 — Landing Page: Brew & Bloom Café

**Track:** Web Development & Design  
**Level:** 1  
**Task:** Landing Page  
**Intern:** Pavani Shetty

## Project Overview

A visually polished landing page for **$hetty's Café**, a fictional artisan coffee shop. Built with warm coffee-themed colors, royalty-free Unsplash images, and a fully responsive layout.

## Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- Google Fonts (Cormorant Garamond + Outfit)

## Sections

- **Hero** — Full-screen background image with headline and CTA
- **Menu** — 4 coffee & pastry items with images and prices
- **About** — Brand story with stats and barista image
- **Testimonials** — 3 customer reviews with profile photos
- **Gallery** — 5 café photos in a masonry-style grid
- **Contact** — Address, hours, and contact form
- **Footer** — Brand info and social links

## Feature Checklist

- [x] Fixed/sticky navigation bar with 5 navigation links
- [x] Hero section with headline, subheadline, and CTA button
- [x] Multiple content sections (Menu, About, Testimonials, Gallery, Contact)
- [x] Footer with contact/social links
- [x] Consistent coffee-themed colour palette
- [x] Responsive layout (Flexbox & Grid)
- [x] No element overlap — intentional padding, margin, box-sizing
- [x] Clean typography with multiple font sizes (Cormorant Garamond + Outfit)

## How to Run

Open `index.html` in any modern web browser.
